
# 语句排序
## 解题原则：

*   排除法

```
A:②
B:②
C:⑤
D:⑤
```

    1. 观察选项，确定首句</br>
    多数留存法（可优先尝试（连贯性验证），如尝试失败，直接排除）
    
        1. ） 可能是首句
                * 援引观点（引用）--->适合引出话题。
                引用--->观点--->反面论证（举例子）
        2. ） 通常不适合做首句 
                * 指代类描述（代词--没有明确的指代对象的时候）
                * 补充类描述（也，还，又）
                * 结论类表述（适合句尾）（因此，于是，终于）
        3. ) 观察选项，找捆绑
                * 话题递归

# 语句衔接（优化表达）
##解题技巧
1. 话题衔接
A,B,C,____。
“横线的内容是与ABC并列，就近原则。因为话题在过渡。”
2. 前后照应（中间）
A，_____,B。
“横线填A+B的句子，承上启下，如果没有A，那么就选启下的话题B”
3. 行文脉络
____,A,B(总分)
"总结"
 A，B。____（分总）
