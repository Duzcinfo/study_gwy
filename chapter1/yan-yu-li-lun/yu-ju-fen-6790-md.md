

# 解题原则：

*   排除法

```
A:②
B:②
C:⑤
D:⑤
```

    1. 观察选项，确定首句</br>
    多数留存法（可优先尝试（连贯性验证），如尝试失败，直接排除）
    
        1. ） 可能是首句
                * 援引观点（引用）--->适合引出话题。
                引用--->观点--->反面论证（举例子）
        2. ） 通常不适合做首句 
                * 指代类描述（代词--没有明确的指代对象的时候）
                * 补充类描述（也，还，又）
                * 结论类表述（适合句尾）（因此，于是，终于）
        3. ) 观察选项，找捆绑
