# 倍数特性

整除型

* 判断方法

```shell
1 . 3/9 看各位   4/8看末2位   2/5 看末尾  【口诀】
2 . 拆分：如721   --> 700+21
3 . 复杂倍数用因式分解
```

余数型

```shell
苹果每人分10个，还剩3个  ---> 10x = y - 3
苹果每人分10个，还差3个  ---> 10x = y + 3
```

比例型

```shell
a : b = m : n （m、n互质）
a 是 m 的倍数
b 是 n 的倍数
a + b 是 m + n 的倍数
a - b 是 m - n 的倍数
```

## 不定方程

#### 未知数是整数

* 奇偶

ax + by =M \( a, b  一奇一偶\)

3x + 4y = 25 求x 。 3x为奇数---&gt; x为3

* 倍数

ax + by = M ,当 a  或 b 与 M 有公因子。

如： 7x + 3y = 60 求 x + y 的最大值？

因为 3y 为三的倍数，60 为三的倍数，所以 7x 也是三的倍数。即 x 取 3、6。分别带入

* 尾数

ax + by =M ,当 a 或 b 尾数是 0 或 5 时。

37x  + 20y =271  x=? \( x,y 均为整数\)

x = 3  y = 8

# 多次相遇

两端出发相遇

第一次相遇，共走 1 s.

第二次相遇，共走 3 s

第 n 次相遇，共走 （ 2n -1 ）s

s总 = （ 2n  - 1 ）s全长

1. s总： 运动工程中，两人所走的路程和 = （ v1 + v2 ） \* T
2. n 为相遇次数
3. s全场为 A 到 B 的距离
