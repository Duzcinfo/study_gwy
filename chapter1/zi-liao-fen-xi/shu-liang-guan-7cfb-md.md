# 倍数特性

整除型

* 判断方法

```shell
1 . 3/9 看各位   4/8看末2位   2/5 看末尾  【口诀】
2 . 拆分：如721   --> 700+21
3 . 复杂倍数用因式分解
```

余数型

```shell
苹果每人分10个，还剩3个  ---> 10x = y - 3
苹果每人分10个，还差3个  ---> 10x = y + 3
```

比例型

```shell
a : b = m : n （m、n互质）
a 是 m 的倍数
b 是 n 的倍数
a + b 是 m + n 的倍数
a - b 是 m - n 的倍数
```
